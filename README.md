# Выпускной альбом 4 «Е» — сайт для GitHub Pages

## Что внутри

- адаптивный одностраничный цифровой фотоальбом;
- полноэкранная обложка, фильтры, фотосетка и lightbox;
- управление клавишами ← → и поддержка смартфонов;
- оптимизированная загрузка WebP;
- без внешних библиотек и платных сервисов;
- все подписи класса находятся в `config.js`.

## Быстрый запуск с фотографиями

1. Распакуйте архив.
2. Скопируйте исходные JPG/PNG/WebP в папку `incoming`.
3. Дважды щёлкните `START_IMPORT_PHOTOS.cmd`.
4. Для проверки запустите `START_LOCAL_SITE.cmd`.

Скрипт автоматически:
- исправит ориентацию по EXIF;
- создаст полноразмерные WebP до 2200 px;
- создаст быстрые миниатюры до 900 px;
- сформирует `assets/gallery-data.js`.

## Публикация на GitHub Pages

1. Создайте новый публичный репозиторий на GitHub.
2. Загрузите в корень репозитория всё содержимое этой папки.
3. Откройте **Settings → Pages**.
4. В **Build and deployment** выберите **Deploy from a branch**.
5. Выберите ветку `main`, папку `/ (root)` и нажмите **Save**.
6. Через несколько минут GitHub покажет адрес сайта.

## Настройка текста

Откройте `config.js` и измените:
- `school`;
- `className`;
- `year`;
- `city`.

## Важно

На GitHub Pages сайт становится общедоступным. Перед публикацией фотографий детей убедитесь, что у вас есть необходимые согласия на размещение.
