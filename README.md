# Новые грани · 4Е — исправленная версия

В этой версии **нет демонстрационных картинок и нет прямых ссылок на вымышленные имена файлов**.
Реальные фотографии скачиваются из публичной папки Mail.ru:

`https://cloud.mail.ru/public/YjPB/rkGWDEVGu`

После первого запуска GitHub Actions фотографии преобразуются в WebP и сохраняются в единственную папку `photos/`. Сайт затем загружает только локальные файлы GitHub Pages.

## Как заменить текущую версию

1. Удалите старые файлы репозитория `NOVIYE-GRANI-4E` или загрузите содержимое этого архива с заменой.
2. Убедитесь, что ветка называется `main`.
3. Откройте вкладку **Actions** и разрешите workflows, если GitHub предложит это сделать.
4. Откройте workflow **«Получить реальные фотографии»** и нажмите **Run workflow**.
5. Дождитесь зелёной отметки. Workflow добавит папку `photos` с локальными WebP-файлами.
6. В **Settings → Pages** оставьте публикацию из ветки `main`, папка `/(root)`.

После обновления GitHub Pages может потребоваться жёсткая перезагрузка страницы: `Ctrl+F5`.

## Структура

```text
index.html
build_photos.py
.nojekyll
.github/workflows/refresh-photos.yml
photos/                         ← создаётся автоматически
```

Для повторного обновления фотографий снова запустите workflow вручную.
