# 4 «Е» — выпускной альбом 2026

Статический адаптивный сайт в спокойной журнальной стилистике, готовый для GitHub Pages.

## Публикация на GitHub Pages

1. Создайте новый публичный репозиторий GitHub.
2. Загрузите в корень репозитория всё содержимое этой папки.
3. Откройте **Settings → Pages**.
4. В **Build and deployment** выберите **Deploy from a branch**.
5. Выберите ветку `main`, папку `/ (root)` и нажмите **Save**.

Сайт будет опубликован по адресу:

`https://ИМЯ-ПОЛЬЗОВАТЕЛЯ.github.io/ИМЯ-РЕПОЗИТОРИЯ/`

## Реальные фотографии

В макете используются выбранные реальные кадры из публичной папки:

`https://cloud.mail.ru/public/YjPB/rkGWDEVGu`

Выбраны файлы серии `IMG_4536`–`IMG_4570`. Они подключены напрямую через публичный сервер превью Облака Mail, поэтому сайт начинает работать сразу после публикации и не раздувает репозиторий.

Публичная ссылка на папку должна оставаться активной.

## Автономная копия фотографий

Для сохранения выбранных кадров внутри сайта запустите на компьютере с интернетом:

```bash
python tools/cache_photos.py
```

Скрипт скачает фотографии в `assets/photos/` и заменит внешние адреса в `index.html` локальными. После этого повторно загрузите файлы в GitHub.
